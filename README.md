# bolognalife-content

Curated content for **Bologna Life**, the civic-dashboard app — a small list of
city-impacting events in Bologna (sports fixtures, BolognaFiere trade shows, major concerts).

The feed is generated by an **n8n workflow** that fetches a handful of sources, curates and
enriches them, validates the result against [`events.schema.json`](events.schema.json), and commits
it here. A GitHub Actions workflow then publishes it to GitHub Pages, so consumers fetch it from a
stable public URL even though this repo stays private.

## What's here

| File | What it is |
|---|---|
| `events.json` | The generated feed: a `meta` envelope + an `events[]` array. Published to Pages on every push to `main`. |
| `events.schema.json` | JSON Schema (draft 2020-12) the feed is validated against before commit. Also published to Pages. |

**Published URLs** (GitHub Pages, once the repo is live under `mabtn`):

- `https://mabtn.github.io/bolognalife-content/events.json`
- `https://mabtn.github.io/bolognalife-content/events.schema.json`

## Format

`events.json` is a single object: a `meta` envelope plus an `events` array. The schema is the source
of truth — the table below is a summary.

### `meta`

| Field | Type | Notes |
|---|---|---|
| `generatedAt` | date-time | When the run produced this document |
| `workflow` | string | Workflow name + semver, e.g. `bolognalife-events@1.0.0` |
| `sources` | string[] | Source identifiers merged into the run (every `provenance.source` must appear here) |
| `itemCount` | integer | Must equal `events.length` |
| `model` | string | LLM used for enrichment, if any (optional) |

### `events[]`

| Field | Type | Notes |
|---|---|---|
| `id` | string | Stable key + dedup key. Convention: `<kind>-<YYYY-MM-DD>-<slug>` |
| `title` | string | Display name |
| `kind` | enum | `soccer` · `basketball` · `baseball` · `football` · `volley` · `fiera` · `concert` · `other` |
| `venue` | string | Where it happens |
| `startDate` | ISO-8601 | Date-only (`"2026-09-15"`) or with time (`"2026-05-18T20:45:00+02:00"`) |
| `endDate` | ISO-8601 \| null | `null` = single-day event |
| `impactLevel` | enum | `low` · `medium` · `high` · `very_high` |
| `description` | string | One short line of context, can be `""` |
| `url` | string | Deep link; `""` or an `http(s)` URL |
| `geo` | object \| null | `{ lat, lng }`, or `null` when geocoding wasn't possible |
| `tags` | string[] | Kebab-case classification tags (LLM-generated), can be `[]` |
| `provenance` | object | `{ source, fetchedAt, confidence }` — audit trail + pipeline confidence (0–1) |
